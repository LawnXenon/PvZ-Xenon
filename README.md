# Plants vs. Zombies Xbox 360 / Android TV Decompilation Project

A work-in-progress decompilation and restoration project focused on the Xbox 360 and Android TV versions of Plants vs. Zombies.

The main objective of this project is to accurately document, reconstruct, and preserve the original game logic, engine behavior, assets, and platform-specific systems while improving portability and long-term maintainability.

At this stage, the project is focused on accuracy and preservation rather than gameplay modifications. Major engine changes, gameplay alterations, or custom features are intentionally avoided until the original game has been fully understood, reconstructed, and stabilized.

## Current Areas of Work

- Native engine reconstruction
- Reverse engineering
- Android JNI restoration
- Platform-specific compatibility
- Rendering and UI restoration
- Asset research and recreation
- Hooking and debugging systems

## Project Status

The project is currently under active development.
